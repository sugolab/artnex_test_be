# ArtNex 플랫폼 MVP 개발 일별 스케줄 상세 내역 (백엔드 중심 3인 시나리오) - 휴식 없는 버전

본 문서는 ArtNex 제조업 AI 마케팅 자동화 SaaS 플랫폼의 PRD, ERD, 기능명세서, 개발명세서를 기반으로, 백엔드 개발자 3인(Back1, Back2, Back3)이 병렬 작업하는 형태의 일별 스케줄을 세부적으로 재구성한 것입니다. 이전 일별 스케줄을 기반으로 각 작업을 하위 태스크로 분해하여 상세히 설명하였습니다. 휴식 없는 버전으로, 주말 및 공휴일 구분 없이 매일 작업을 진행하여 프로젝트 기간을 최적화하였습니다. 개발 기간(49일)을 달력일 기준으로 적용하며, AI 연동(GPT API, Ideogram API, SNS 크롤링) 및 데이터 처리(PostgreSQL, AWS Lambda)를 중점으로 배분. 총 기간: 2025-10-23 ~ 2025-12-15 (개발 49일, 테스트/배포 5일). 

## 1. 프로젝트 개요
- **프로젝트명**: ArtNex 플랫폼 MVP 개발
- **목표**: PRD 기준 MVP 버전 개발 (AI 기반 브랜드 진단, 디자인 생성, 마케팅 실행, 리포트 분석 자동화 원스톱 플랫폼 구축, UI/UX 단순화)
- **개발 인력**: 백엔드 3인 
  - Back1: 브랜드 관리(Brand Managing) 및 콘텐츠 트래킹(Content Tracking) 중심 (KPI 자동 계산, SNS 크롤링 API)
  - Back2: 브랜드 인사이트(Brand Insight) 및 리포트(Brand Report) 중심 (GPT 기반 시장 분석, PDF Export)
  - Back3: 디자인/숏폼 스튜디오(Design/Shortform Studio) 및 캠페인(Campaign) 중심 (Ideogram API 연동, 성과 예측 AI)
- **특이 사항**: 휴식 없는 버전으로 재구성. 병렬 개발 효율화로 모든 개발자가 매일 작업 참여. ERD 기반 테이블(Brands, Brand_Reports 등) 연동 강조. AI 응답 시간(5초 이내) 및 보안(RBAC, JWT) 준수. 달력일 계산 시 주말 포함 (공휴일 무시: 크리스마스 2025-12-25 이후로 영향 없음).
